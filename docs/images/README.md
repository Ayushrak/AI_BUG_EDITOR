Place architecture and screenshot images here so they render in the project README.

Expected filenames (used by README.md):

- `architecture.png` — high-level architecture diagram
- `prs-dashboard.png` — pull-requests / risk dashboard screenshot

Add images (PNG or JPG) to this folder and commit. Example:

```bash
mkdir -p docs/images
# copy your files into docs/images/
git add docs/images/architecture.png docs/images/prs-dashboard.png
git commit -m "Add architecture and dashboard images"
git push
```
