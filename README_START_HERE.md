## 🔧 Quick Start Commands

### 1. Create Project Structure

```bash
mkdir -p codeguardian-ai
cd codeguardian-ai

# Copy the restructuring guide
# Follow directory structure section
```

### 2. Clone & Reorganize Current Code

```bash
# Move your current frontend and backend
cp -r /path/to/frontend ./frontend
cp -r /path/to/backend ./backend

# Reorganize backend into microservices
# (See PROJECT_RESTRUCTURING_GUIDE.md)
```

### 3. Setup Local Environment

```bash
# Copy environment template
cp .env.example .env

# Edit .env with your settings
nano .env

# Start all services
docker-compose up -d

# Verify services
docker-compose ps
```

### 4. Setup Databases

```bash
# Run schema
psql -U postgres -d codeguardian < backend/database/schema.sql

# Test Redis
redis-cli ping  # Should return PONG

# Test Kafka (optional)
kafka-console-producer --topic test --bootstrap-server localhost:9092
```
